# ChatMemo
壁打ちアプリ

## 内容
１日のルーティンを記録したり、自分の発言が相手から見たらどんなふうに感じるのか確認したり、テンプレートを用いて面接の練習をしたり。。。

## 機能
* 一人チャット機能
* 返信テンプレートを用いた自動返信機能
 
## 言語
Kotlin

## アーキテクチャ
MVVM

## 主な使用技術
###  ネイティブ(ライブラリ)
* Koin
* Coroutine
* Room
* LiveData
* DataBinding
* swipereveallayout

### バックエンド
* なし
